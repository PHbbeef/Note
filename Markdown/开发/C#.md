# 报错指令
```bash
# No usable version of the libssl was found
方案：安装libssl1.0.0版本。
我安装的是 libssl1.0.0_1.0.2g-1ubuntu4.20_amd64.deb
找到你要的版本 http://security.ubuntu.com/ubuntu/pool/main/o/openssl/
```
