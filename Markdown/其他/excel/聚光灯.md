## 聚光灯的实现

+ F9刷新
+ Ctrl+j（代码提示）

| OR     |            |               |
| :----- | :--------- | :------------ |
| ROW    | 返回单元格所选的行数 |               |
| COLUMN | 返回单元格所选的列数 |               |
| CELL   | 返回引用数      | =CELL("col") |
| OR   | 任何一个逻辑为真则为true     |  |


+ 全选表格`条件格式`,选择`新建规则`，`使用公式确定单元格格式`
+ 输入公式`=OR(ROW()=CELL("row"),COLUMN()=CELL("col"))`
+ 条件格式选择颜色

+ 为单元格新建一个VBA
    + `通用`改为`Worksheet`然后会有一个函数生成，添加`Calculate`
	

+ 使用Calculate函数可以在VBA代码中手动触发工作表的计算,而不必等待Excel自动计算
